# Dollar General (dollar-general)

Dollar General is a Fortune 500 discount retailer operating more than 20,000 stores across the United States. This repository captures Dollar General's developer-facing surfaces, partner channels, and vendor portals. Dollar General does not currently publish a public REST API; vendor and supplier integration runs through the DG Partners portal and RangeMe for new vendor onboarding.
